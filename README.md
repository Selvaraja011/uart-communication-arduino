# UART Serial Communication — Arduino Uno

## 🔗 Live Simulation
▶️ https://wokwi.com/projects/460262121393958913

---

## 📌 Project Overview
This project demonstrates UART (Universal Asynchronous 
Receiver-Transmitter) serial communication using 
Arduino Uno, simulated on Wokwi online platform.

---

## ⚙️ How UART Works
| Parameter | Value |
|-----------|-------|
| Baud Rate | 9600 bps |
| Data Bits | 8 |
| Stop Bits | 1 |
| Parity    | None |
| Mode      | Full Duplex |

---

## 🛠️ Components Used
- Arduino Uno (Simulated)
- Built-in LED (Pin 13)
- Serial Monitor (UART Terminal)

---

## 💡 Features
- Periodic UART data transmission every 2 seconds
- Real-time data reception from Serial Monitor
- Echo back received data with byte count
- LED blink indication for TX and RX events
- Message counter tracking

## ▶️ How to Run
1. Click the Wokwi Live Link above
2. Press ▶️ Play button
3. Open **Serial Monitor** at bottom
4. Watch TX messages appear every 2 seconds
5. Type any message → Press Enter → See RX echo
